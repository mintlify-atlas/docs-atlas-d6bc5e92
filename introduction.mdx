Welcome to the WooCommerce Azul Payment Gateway documentation. This plugin enables seamless integration between your WooCommerce store and Azul, the payment processing service provided by Banco Popular Dominicano.

## What is the Azul payment gateway?

The WooCommerce Azul Payment Gateway is a WordPress plugin that allows you to accept credit and debit card payments through Azul's secure payment platform. It provides a simple yet powerful integration that handles payment processing, transaction validation, and order management.

## How it works

The payment flow is straightforward:

1. Customer selects Azul as their payment method at checkout
2. Order details are encrypted with SHA-512 authentication hash
3. Customer is redirected to Azul's secure payment page
4. Customer enters their card information on Azul's platform
5. Azul processes the payment and returns the customer to your store
6. Plugin validates the response and updates the order status accordingly

<Info>
The plugin never handles sensitive card data directly. All payment information is entered on Azul's PCI-compliant payment page.
</Info>

## Technical overview

The plugin consists of three main components:

### Payment gateway class

Extends `WC_Payment_Gateway` to integrate with WooCommerce's payment processing system. Handles configuration, form fields, and payment initiation.

### Process template

A WordPress page template that receives order data and automatically submits it to Azul's payment page via a POST form.

### Response handler

Processes callbacks from Azul, validates payment responses, and updates WooCommerce order statuses.

<Note>
This plugin requires WooCommerce to be installed and activated on your WordPress site.
</Note>
